# Kho Templates

Thư mục này chứa các template tham khảo cho từng format output.

## Cấu trúc

```
templates/
├── slide/          ← Template cho slide / slideshow
├── pdf/            ← Template cho PDF (sẵn sàng)
└── docs/           ← Template cho Google Docs / Word (sẵn sàng)
```

## Cách dùng

1. Chọn template phù hợp với format cần tạo
2. Đọc hướng dẫn tương ứng trong `formats/` để biết lấy content từ đâu
3. Áp dụng template + content → output
