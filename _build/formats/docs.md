# Hướng dẫn tạo Google Docs / Word

## Mục đích

Tạo tài liệu có thể chỉnh sửa cho:
- Chia sẻ nội bộ team (comment, suggest)
- Gửi review cho stakeholder
- Base document cho các format khác

## Cách trích xuất

Copy toàn bộ nội dung markdown, paste vào Google Docs / Word. Điều chỉnh:

| Markdown | Google Docs / Word |
|----------|-------------------|
| `# Heading 1` | Heading 1 style |
| `## Heading 2` | Heading 2 style |
| `> Blockquote` | Italic hoặc highlight box |
| `<!-- Comment -->` | Xoá (không cần trong docs) |
| `<img src="...">` | Insert image từ `screenshot/` |
| Bảng markdown | Chuyển sang table |

## Cấu trúc document gợi ý

```
1. Title page
2. Mục lục (auto-generated từ headings)
3. Giới thiệu (gioi-thieu/)
4. Tính năng (tinh-nang/ — mỗi file = 1 section)
5. Sắp ra mắt (sap-ra-mat/)
6. Phụ lục: Screenshots
```

## Quy ước

- **Font:** Be Vietnam Pro 11pt hoặc Arial 11pt
- **Line spacing:** 1.5
- **Headings:** Bold, theo hierarchy
- **Screenshots:** Inline, max-width 60% page
- **Page break:** Trước mỗi tính năng mới
